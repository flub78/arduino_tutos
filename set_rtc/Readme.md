# Simple Alarm Clock

- based on RTC SD3231
- 4 buttons
- one LCD
- One buzzer

## Use cases

- display date and time
- change display mode (mode)
- change settings (mode long + mode + up or down)
- toggle alarm (clear)

when ringing flashing
	stop the alarm 
	
when ringing/flashing
	again in 5 mins
	
## Buttons

	- Mode
	- Up and down
	- Clear/Esc
